# photoninja


This repo contains clone of Unsplash.
To run this on your local.


1. Fork this repo


2. Clone it.


3. run npm install in root directory


4. run "npm run dev" to run your project in local.  


Features:
1. Search
2. Infinite Scroll
3. View Photos
